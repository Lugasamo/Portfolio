LUIS SAAVEDRA PORTFOLIO — UPDATED STATIC EXPORT

This is a plain HTML/CSS/image website exported from the locally restored Grav
site after the latest Admin updates. It does not require Grav, PHP, a database,
Composer, or Node.js on the web server.

To publish, upload all contents of this folder to the root of a static host.
Keep index.html in the root directory.

The old Grav contact form requires a static form provider before submissions
can be delivered. Netlify Forms or Formspree can provide this service.
